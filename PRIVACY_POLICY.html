Política de Privacidade - Cadrimil

Última atualização: 01 de Dezembro de 2024

A sua privacidade é importante para nós. Esta política de privacidade explica como o aplicativo Cadrimil coleta, usa e protege as suas informações.

1. Coleta de Dados

1.1. Dados Fornecidos pelo Usuário

O Cadrimil não exige a criação de uma conta de usuário para ser utilizado. Todas as informações inseridas no aplicativo, como dados de missões, datas e valores, são armazenadas localmente no seu dispositivo e não são enviadas para nenhum servidor externo de nossa propriedade.

1.2. Dados Coletados Automaticamente (Publicidade)

O aplicativo utiliza o Google AdMob para exibir anúncios. O Google AdMob pode coletar e usar dados do seu dispositivo, incluindo:

  • Identificadores de publicidade (como o ID de publicidade do Google).
  • Informações sobre o dispositivo (modelo, sistema operacional).
  • Dados de uso e interação com anúncios.

Esses dados são utilizados exclusivamente para fornecer anúncios relevantes e analisar o desempenho da publicidade. Para mais informações, consulte a Política de Privacidade do Google.

2. Uso das Informações

As informações inseridas por você no aplicativo são utilizadas exclusivamente para:

  • Realizar os cálculos de diárias solicitados.
  • Gerar relatórios em PDF (quando solicitado).
  • Salvar seu histórico de missões no próprio dispositivo.

3. Compartilhamento de Dados

3.1. Exportação de Dados

O aplicativo permite que você exporte seus dados de missões (em formato JSON ou PDF) para compartilhamento. Essa ação é iniciada exclusivamente por você, e os dados são enviados através dos aplicativos que você escolher (ex: WhatsApp, E-mail).

3.2. Terceiros

Não compartilhamos suas informações pessoais ou dados de missões com terceiros, exceto os dados técnicos coletados automaticamente pelo Google AdMob conforme descrito na seção 1.2.

4. Armazenamento e Segurança

Seus dados de missões são armazenados localmente no seu dispositivo utilizando tecnologias de persistência segura (AsyncStorage). Recomendamos que você mantenha seu dispositivo seguro e protegido por senha.

5. Seus Direitos

Como os dados são armazenados localmente:

  • Acesso e Correção: Você pode visualizar e editar todas as suas informações diretamente no aplicativo.
  • Exclusão: Você pode excluir qualquer missão ou período diretamente na interface do aplicativo. Para excluir todos os dados, basta limpar os dados de armazenamento do aplicativo nas configurações do seu dispositivo ou desinstalar o app.

6. Alterações nesta Política

Podemos atualizar nossa Política de Privacidade periodicamente. Recomendamos que você revise esta página regularmente para verificar quaisquer alterações. As alterações entram em vigor imediatamente após serem publicadas nesta página.
